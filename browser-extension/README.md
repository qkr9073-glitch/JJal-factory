# 짤공장 릴스 헌터 수집기 (Chrome 확장)

인스타 **릴스 + 이미지 게시물**, 유튜브 쇼츠의 URL·조회수·좋아요·**이미지 원본**을 모아
짤공장 서버(`http://localhost:8777`)로 보내는 로컬 확장입니다. **로그인·API 안 씀** — 그냥 둘러보면 네트워크 응답을 읽어 수집.

## 설치
1. 짤공장 서버를 먼저 켜둡니다 (`http://localhost:8777` 접속됨 확인).
2. Chrome `chrome://extensions` → **개발자 모드 ON**.
3. **압축해제된 확장 프로그램을 로드** → 이 `browser-extension` 폴더 선택.

## 사용
1. 인스타(릴스탭/프로필/탐색) 또는 유튜브 쇼츠로 들어갑니다.
2. 확장 아이콘 → **현재 탭 감시 시작**.
3. 화면을 **스크롤**하면 릴스·이미지 게시물이 자동 수집됩니다.
4. **짤공장으로 보내기** → 서버로 전송.
5. 짤공장 → **수입·수출 → 🧲 브라우저 수집**에서 확인 → ＋소재 → 템플릿 → 만들기.

## 참고
- Chrome 상단 "디버깅 중" 배너는 정상(네트워크 감청 방식).
- 폴더를 옮기면 `chrome://extensions`에서 재로드 필요.
- 이미지 게시물은 이미지 URL이 응답에 들어있어 바로 짤 소재로 쓸 수 있고, 릴스는 URL·조회수만 수집(영상 자동제작은 서버측 준비 중).
- 인스타 응답 구조는 버전마다 달라, 이미지 캐러셀 수집이 안 잡히면 background.js의 `normalizeInstagramNode`/`igImageUrls`를 실제 응답 보고 튜닝해야 함.
