# 짤공장 사용법 (1분 요약)

## 직원용 웹서버 (B안)
1. `START-SERVER.cmd` 더블클릭 → 서버(8777) + 공개 터널 동시 시작
2. 공개 주소 확인: `TUNNEL-URL.cmd` 더블클릭 (재시작할 때마다 주소가 바뀜에 주의)
3. 직원 사용법: 주소 접속 → 접속코드(config.json의 access_code, 기본 7777) 입력 →
   - 🔥 인기글 불러오기: 8개 커뮤(디시실베/루리웹/에펨포텐/개드립/더쿠/인벤/인스티즈/웃긴대학) 인기글 목록
     → 커뮤별·주제별 필터 + 추천/조회/댓글/최신순 정렬 → [만들기] 클릭
   - 또는 링크 직접 붙여넣기 → 완성팩 만들기
   - 결과: 썸네일 3종 중 선택 + zip 다운로드 + 본문 복사
4. 부팅 시 자동시작 등록(관리자 아님, 1회 — 직접 실행 필요):
   `schtasks /Create /TN "MemeFactoryServer" /TR "E:\짤공장 (게시물자동화)\START-SERVER.cmd" /SC ONLOGON /F`
5. 에펨은 잦은 요청 시 일시 차단(430)될 수 있음 — 10분 캐시가 있어 보통 문제없고, 자연 해제됨


## 준비 (최초 1회)
1. https://aistudio.google.com/apikey → API 키 만들기 (구글 로그인만 하면 됨, 무료)
2. `config.json` 열어서 붙여넣기:
   - `"gemini_api_key": "AIza..."`
   - `"watermark": "@내계정핸들"`

## 평소 쓰는 법

### ① 링크로 만들기 (주력)
1. 디시/루리웹/에펨 등에서 웃긴 글 발견 → 주소 복사 (모바일 주소도 OK)
2. **MAKE-URL.cmd 더블클릭**
3. 끝나면 review.html이 자동으로 열림 → 썸네일/짤 확인, [캡션 복사] 버튼
4. 인스타 앱/웹에서: thumb.jpg 첫 장 → 01.jpg~ 순서로 캐러셀 업로드 → 캡션 붙여넣기

### ② 캡처 짤로 만들기 (링크 추출이 안 될 때)
- 스크린샷 파일들(또는 폴더)을 **MAKE-IMAGES.cmd에 드래그&드롭**
- 첫 번째 짤이 썸네일이 되니까 제일 웃긴 캡처를 1번으로

## 결과물 위치
`결과물\날짜_제목\` — thumb.jpg / 01.jpg~ / caption.txt / review.html

## 문제 생기면
- `python make.py <URL> --debug` 로 상세 오류 확인
- AI가 "skip 판정" 내리면 이유 보고 직접 판단 (정치/수위/참사조롱은 올리지 말 것)
