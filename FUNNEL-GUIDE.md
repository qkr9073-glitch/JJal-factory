# 댓글 → DM 전자책 퍼널 세팅 가이드

카드뉴스 공장이 만들어주는 것: **캐러셀 이미지 + 캡션(댓글 키워드 CTA 포함) + 전자책 PDF**.
이 문서는 나머지 반쪽 — "댓글 달면 DM으로 PDF 자동 전송"을 붙이는 방법.

## 전체 흐름

```
캐러셀 업로드 → 시청자가 댓글에 키워드 입력 (예: "후킹")
→ ManyChat이 감지 → 공개 대댓글 + DM 자동 전송 (PDF 링크 버튼)
→ (선택) 팔로우 체크 후 전송 → 리드 확보
```

## 1) 준비물

| 항목 | 내용 |
|---|---|
| 인스타 계정 | **프로페셔널(크리에이터/비즈니스) 계정** 필수 — 설정에서 전환 |
| 자동화 툴 | **ManyChat** (manychat.com) — 댓글 트리거 DM은 Meta 공식 API 기반이라 정지 위험 없음 |
| PDF 링크 | 아래 2) 참고 |

ManyChat 무료 플랜: 컨택 1,000명까지 / 기본 댓글→DM 자동화 가능. 넘치면 Pro(월 $15~).

## 2) PDF 호스팅 (링크 만들기)

DM에는 파일이 아니라 **링크**를 보낸다. 옵션:

| 방법 | 장점 | 단점 | 추천 |
|---|---|---|---|
| **Google Drive 공유 링크** | 1분 세팅, 무료 | 링크가 못생김, 브랜딩 없음 | ⭐ 시작용 |
| **Cloudflare Pages** (보조계정 활용) | 자체 URL, 빠름, 무제한 무료 | 업로드가 한 단계 더 | 반응 좋아지면 |
| 리틀리/스탠스토어 | 이메일 수집 겸용 | 외부 의존 | 이메일 리스트 욕심나면 |

드라이브: 업로드 → 우클릭 → 공유 → "링크가 있는 모든 사용자" → 링크 복사. 끝.

## 3) ManyChat 세팅 (10분, 게시물마다 반복은 3분)

1. manychat.com 가입 → **Instagram 연결** (인스타 로그인 + 권한 허용)
2. 좌측 **Automation → New Automation → Instagram → "User comments on your Post or Reel"**
3. 트리거 설정:
   - 게시물: **특정 게시물 선택** (방금 올린 카드뉴스)
   - 키워드: 캡션에 쓴 키워드와 동일하게 (예: `후킹`) — "contains" 조건
4. 응답 설정 (순서대로 두 단계):
   - **Public Reply(공개 대댓글)**: "DM 보냈어요! 확인해 주세요 📩" — 문구 2~3개 등록해 랜덤 발송(스팸 감지 회피)
   - **DM 메시지**: 인사 한 줄 + **버튼(링크)** = PDF 링크. 예:
     > 요청하신 「쇼츠 후킹 문장 60」 PDF예요 🎁
     > 아래 버튼 눌러 받아가세요!
     > [📕 PDF 받기] ← 버튼에 링크
5. (선택) **팔로우 체크**: DM 첫 단계에 "팔로우 하셨나요?" 버튼 분기 → 팔로워 전환율↑
6. **Set Live** 누르면 가동.

## 4) 게시물 올릴 때마다 하는 일 (운영 루틴)

1. `/card`에서 완성팩 생성 → 캐러셀 업로드 + 캡션 붙여넣기
2. PDF를 드라이브에 올리고 링크 복사
3. ManyChat 자동화 복제(Duplicate) → 게시물/키워드/링크 3개만 교체 → Live
4. 부계정으로 댓글 테스트 → DM 수신 확인

## 5) 주의사항

- **키워드 불일치가 최다 사고** — 캡션 키워드와 ManyChat 키워드 철자 확인 (완성팩 review.html에 키워드 표기됨)
- 24시간 룰: 댓글 트리거 DM은 허용되지만, 그 외 임의 DM 대량 발송은 금지
- 공개 대댓글 문구는 주기적으로 바꿔줄 것 (동일 문구 반복 = 스팸 신호)
- PDF 내용의 저작권은 본인 책임 — 생성 후 한 번 훑어보고 이상한 항목은 items.json 참고해 수정 재생성

## 6) 나중에 (선택 업그레이드)

- Meta Graph API 직접 구현으로 ManyChat 탈피 (앱 리뷰 필요, 초기엔 비추천)
- Cloudflare Pages에 PDF 랜딩 페이지 + 다운로드 수 카운트
- DM에서 이메일 받고 뉴스레터 연결 (강의 퍼널로 연장)
